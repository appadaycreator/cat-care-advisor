# 猫の飼育・ケア診断【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Cat Care Advisor
**バージョン**: 1.3.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/cat-care-advisor/

猫の年齢・品種・生活環境から最適なケア方法と必要な費用を診断。登録不要・完全無料でご利用いただけます。

## v1.3.0 での改善内容（P1・P2実装完了）

### P1: 結果表示時の文脈CTAボタン設置（M1改善）
- **おすすめ商品セクション**: 診断結果下に「このプランにおすすめの商品」セクション追加
  - 子猫/成猫/シニア/老猫の4プラン各3件の関連商品を動的表示
  - フード、サプリメント、ケア用品などを結果に応じて推薦
  - A8.netアフィリエイトリンク統合で収益性向上

### P2: クイズ入力値の永続化と復元（M2・M3改善）
- **診断進捗の自動保存**: 各質問への回答をlocalStorageに自動保存
  - ページ離脱時に進捗が失われず、再訪問時に復元可能
  - 24時間以内の前回診断に対して「診断を再開しますか？」確認ダイアログ表示
- **ユーザー体験向上**: 途中離脱→再開のシームレスな体験

## v1.2.0 での改善内容（P1・P2実装完了）

### P1: データ保存・履歴機能（M3）
- **localStorage 自動保存**: 診断結果をブラウザのローカルストレージに自動保存
  - 同じデバイス・ブラウザなら前回結果を自動復元
  - タイムスタンプ付きで保存日時を記録
- **プライバシー保護**: 個人情報・入力データはサーバー送信なし、すべてローカル処理

### P2: グラフ・可視化（M4）
- **Chart.js による費用内訳可視化**: 診断結果に月額費用の内訳ドーナッツグラフを表示
  - 食費・医療費・生活用品などを視覚的に比較
  - レスポンシブ対応で PC・スマホ両対応
  - 色分けで各項目を識別しやすく

### P3: 印刷・エクスポート機能（M10）
- **印刷ボタン**: 診断結果を印刷用フォーマットで出力
  - print.css で見出し・ボタン・広告を非表示化
  - 診断結果のみを見やすく印刷
- **CSV エクスポート**: 診断結果をダウンロード可能に
  - 費用内訳をテーブル形式で CSV として出力
  - Excel・Google Sheets で開いて分析可能
  - 日時スタンプ付きファイル名で複数結果を管理

## v1.1.0 での改善内容（P1・P2実装完了）

### P1: 費用内訳の詳細化と URL シェア機能
- **費用内訳タブ**: 各ケアプランに内訳テーブルを追加
  - 食費、健診・検査、生活用品、医療費などを項目別に表示
  - 月額・年額・5年費用を自動計算
- **URL シェア機能**: 診断結果を URLパラメータで共有可能に
  - 例: `?result=adult` で成猫ケアプラン結果を直接表示
  - SNS シェア時に結果URL を自動生成
  - 「URLをコピー」ボタンで簡単シェア
- **SNS シェアテキスト**: 各結果に適切な絵文字とタイトル付きテキストを自動生成

### P2: モバイル UX 最適化
- **ボタンサイズ改善**: すべてのボタンを最小 48px 高さに統一（WCAG AA準拠）
- **選択肢ボタン**: パディング・行高拡大、最小 52px 高さ確保
- **質問テキスト**: モバイル時のフォントサイズアップ（18px）、行間拡大
- **タッチ対象拡大**: フォーム要素の最小高さを 48px に（誤タップ防止）
- **間隔最適化**: モバイルでのカード・マージン調整

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: 猫の飼育・ケア診断は無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

はい、結果はブラウザのローカルストレージに自動保存されます。同じデバイスなら前回の結果を復元できます。また、CSV形式でダウンロードして管理することもできます。

**Q: 結果を印刷・エクスポートできますか？**

はい、印刷ボタン（🖨️）または CSV エクスポートボタン（📥）で結果を出力できます。CSV は Excel・Google Sheets で分析可能です。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
